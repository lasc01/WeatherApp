I work as a web designer/developer in Lagos, Nigeria. I'm passionate about assisting individuals in creating astounding websites for web and mobile devices. I discovered my affinity for coding after having earned a Bsc. in Physics from the Federal University of Agriculture in Abeokuta. Since then, I've acquired a plethora of experience designing and developing web and mobile applications. My first project as a web designer was in April of 2019, when I designed a website for "DJ Lasc Music Empire" using WordPress. I then continued by enrolling in some online courses on AQskills and Udemy in order to improve my web design skills and also learn web development. Since then, I've been assisting businesses and companies in designing and developing fantastic websites for web and mobile devices.
